# Codex Phase 0 UI Prompt

You are Codex working on LandscapeOps Phase 0. Read `AGENTS.md`, `CODEX.md`, the design docs, scheduling docs, reporting docs, onboarding/settings docs, and build plan before changing UI.

Your job is to make the initial app shell feel premium, operational, and specific to landscape contractors. Focus on:

- Mobile-first layout and responsive dashboard density.
- Dark graphite/navy navigation with warm content surfaces.
- Muted landscape greens, brass accents, polished cards, and clear status colors.
- Accessible buttons, landmarks, labels, focus states, and readable contrast.
- Real module language for leads, estimates, jobs, schedule, field ops, invoices, and reports.

Do not make a generic marketing landing page. The first screen should feel like a working SaaS product preview.
