# Design System

LandscapeOps should feel premium, modern, operational, and specific to landscape contractors.

## Visual Direction

- Base: dark graphite and navy for navigation and executive surfaces.
- Content: warm off-white, linen, and cream work surfaces.
- Accents: muted landscape greens and restrained brass/gold.
- Status: green for healthy, amber for attention, blue for queued/informational, red for blocked or overdue.

## UI Principles

- Mobile-first layouts with strong touch targets.
- Dense but readable dashboard content.
- Polished cards for repeated objects and metrics.
- Clear focus states and keyboard-accessible controls.
- Professional typography with no decorative gimmicks.

## Component Direction

Use shadcn/ui conventions when adding reusable primitives. Keep radius modest, spacing intentional, and labels operational. Avoid generic SaaS filler text and overly playful decoration.
