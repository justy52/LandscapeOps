# Premium UI Direction

The app should look expensive and state-of-the-art without becoming flashy. The feel is executive operations, not a marketing page.

## Shell

Use a dark sidebar/topbar base with warm, high-contrast content areas. Navigation should include Dashboard, Leads, Estimates, Jobs, Schedule, Field Ops, Invoices, Reports, and later Settings.

## Dashboard

The first dashboard should surface pipeline, pending estimates, active jobs, overdue AR, crew schedule, and module entry points. Cards should have strong spacing, subtle borders, and clear hierarchy.

## Mobile

Mobile should not be an afterthought. Navigation can become a horizontal strip or bottom-oriented pattern later. Cards should stack cleanly, buttons should be easy to tap, and text should remain readable without crowding.

## Empty States

Empty states should be confident and useful. They should explain the operational state, not apologize. Avoid cartoonish illustration; use refined iconography and direct calls to action.
