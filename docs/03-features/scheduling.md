# Scheduling

Scheduling coordinates crews, equipment, site visits, installs, maintenance routes, subcontractors, weather constraints, and customer commitments.

## Calendar Direction

The calendar should support day, week, route, and crew views in later phases. Phase 0 only previews the schedule concept in the dashboard shell.

## Scheduling Data

Future scheduling models should connect jobs, customers, crews, users, equipment, visits, task lists, and location data. Each schedule item must remain org-scoped and auditable when changed.

## UX Priorities

- Mobile-friendly crew view for today.
- Clear status colors for on track, watch, delayed, complete, and blocked.
- Fast rescheduling without losing the reason for the change.
- Route pressure and workload density visible to operations managers.
